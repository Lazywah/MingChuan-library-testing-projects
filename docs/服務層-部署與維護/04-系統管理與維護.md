# 04 - 系統管理與維護 | System Administration & Maintenance

## 資料庫備份 | Database Backup

### 手動備份 (PowerShell)
```powershell
# ZH: 複製 SQLite 資料庫檔案 | EN: Copy SQLite database file
$timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
Copy-Item data\ai_platform.db -Destination "backups\ai_platform_$timestamp.db"
```

### 排程備份 (Windows 工作排程器)
建議建立一個 PowerShell 腳本 `backup.ps1`：
```powershell
$timestamp = Get-Date -Format "yyyyMMdd"
Copy-Item C:\CodeSpace\data\ai_platform.db -Destination "C:\CodeSpace\backups\daily_$timestamp.db"
```
然後在「Windows 工作排程器 (Task Scheduler)」中設定每天凌晨 3 點執行此腳本。

### 資料庫完全重置 (Destructive Reset)
> [!CAUTION]
> 此操作將永久刪除所有使用者、任務與對話紀錄。執行前請務必確認已完成資料備份。

當系統資料異常或需要重新開始時使用：
1. **停止服務**：`docker-compose stop job-scheduler`
2. **刪除檔案** (PowerShell)：`Remove-Item data\ai_platform.db*`
3. **重啟服務**：`docker-compose up -d job-scheduler`
（系統啟動時會自動根據 `models.py` 重新建立空白資料表）

## 日誌查看 | Log Viewing

```bash
# 所有服務日誌
docker-compose logs

# 特定服務 (即時追蹤)
docker-compose logs -f job-scheduler
docker-compose logs -f open-webui

# 最近 100 行
docker-compose logs --tail=100 job-scheduler
```

## 系統監控 | System Monitoring

```bash
# 容器狀態
docker-compose ps

# 資源使用量
docker stats

# 健康檢查
curl http://localhost:8002/health
```

## 資料庫查閱 | Database Inspection

### 推薦工具
建議下載 `data/ai_platform.db` 後使用 **DB Browser for SQLite**。

### 常用查詢指令 (透過 Docker)
```bash
# 查看所有帳號
docker exec ai-platform-scheduler python -c "import sqlite3; conn = sqlite3.connect('/data/ai_platform.db'); [print(row) for row in conn.execute('SELECT username, role FROM users').fetchall()]"

# 查看任務佇列情況
docker exec ai-platform-scheduler python -c "import sqlite3; conn = sqlite3.connect('/data/ai_platform.db'); [print(row) for row in conn.execute('SELECT job_id, status, progress FROM training_jobs ORDER BY created_at DESC LIMIT 5').fetchall()]"
```

## Token 管理 | Token Management

Token 會在每月第 `TOKEN_RESET_DAY` 天自動重置。

### 手動將所有使用者的 Token 消耗歸零 (Reset all users' tokens to 0)
```bash
docker exec ai-platform-scheduler python -c "import sqlite3; conn = sqlite3.connect('/data/ai_platform.db'); conn.execute('UPDATE token_usage SET tokens_used = 0'); conn.commit()"
```

### 手動調整用戶額度
```bash
docker exec ai-platform-scheduler python -c "import sqlite3; conn = sqlite3.connect('/data/ai_platform.db'); conn.execute('UPDATE token_usage SET tokens_limit=10000000 WHERE user_id=\"USER_ID\"'); conn.commit()"
```
