# 04 - 系統管理與維護 | System Administration & Maintenance

## 資料庫備份 | Database Backup

### 手動備份
```bash
# ZH: 複製 SQLite 資料庫檔案 | EN: Copy SQLite database file
cp data/ai_platform.db backups/ai_platform_$(date +%Y%m%d_%H%M%S).db
```

### 排程備份 (Linux crontab)
```bash
# 每天凌晨 3 點備份
0 3 * * * cp /opt/ai-platform/data/ai_platform.db /opt/ai-platform/backups/daily_$(date +\%Y\%m\%d).db
```

### 還原
```bash
docker-compose down
cp backups/ai_platform_YYYYMMDD.db data/ai_platform.db
docker-compose up -d
```

## 日誌查看 | Log Viewing

```bash
# 所有服務日誌
docker-compose logs

# 特定服務 (即時追蹤)
docker-compose logs -f job-scheduler
docker-compose logs -f open-webui

# 最近 100 行
docker-compose logs --tail=100 job-scheduler
```

## 系統監控 | System Monitoring

```bash
# 容器狀態
docker-compose ps

# 資源使用量
docker stats

# 健康檢查
curl http://localhost:8002/health
```

## Token 管理 | Token Management

Token 會在每月第 `TOKEN_RESET_DAY` 天自動重置。

### 手動調整用戶額度
```bash
# 透過 SQLite 命令列
docker exec -it ai-platform-scheduler sh -c \
  "python -c \"
import sqlite3
conn = sqlite3.connect('/data/ai_platform.db')
conn.execute('UPDATE token_usage SET tokens_limit=10000000 WHERE user_id=?', ('USER_ID',))
conn.commit()
\""
```

## 儲存空間 | Storage

| 目錄 | 用途 | 預估大小 |
|------|------|----------|
| `data/` | SQLite 資料庫 | < 100MB |
| Docker volumes | Open WebUI + Ollama 模型 | 5-50GB |

### 清理 Docker 暫存
```bash
docker system prune -f
```
